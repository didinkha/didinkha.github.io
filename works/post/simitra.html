Project Description

Pada tahun 2016, saya berkesempatan untuk magang di PT. Bukit Asam, Tbk. Unit Pelabuhan Tarahan, Lampung. Selama di sana saya ditugaskan untuk mengembangkan sebuah sistem informasi bernama "SI Mitra PTBA" sebagai full-stack developer di mana saya harus mengembangkan website baik dari sisi front-end maupun back-end. Saat itu pengetahuan saya terkait PHP masih minim sehingga saya harus belajar secara otodidak selama kurang lebih satu minggu. Setelah itu, saya berhasil mengembangkan sistem ini menggunakan PHP pure (tanpa framework). Lebih lanjut mengenai sistem ini, sistem ini merupakan pengembangan dari sistem kemitraan yang sebelumnya telah digunakan para karyawan. Sistem ini dibangun untuk menangani kendala-kendala dari sistem yang lama seperti kurangnya automasi, validasi, akurasi, dll.
