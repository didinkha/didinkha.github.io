Deskripsi Proyek

Website ini dikembangkan untuk Akademi Ilmu Komputer Ternate. Adapun back-end yang digunakan dalam pengembangan website ini adalah CodeIgniter. Untuk front-end saya menggunakan Bootstrap 4 (from scratch) dan dipadukan dengan framework javascript seperti JQuery, JQuery Confirm, dll.
