Project Description

Di awal tahun 2020, saya berkesempatan untuk mengerjakan sebuah proyek pengembangan aplikasi bagi sebuah perusahaan kelapa sawit di Bontang tepatnya PT. Etam Anugerah Ilahi. Aplikasi dibangun menggunakan teknologi PHP Desktop di mana backend dari aplikasi tersebut sepenuhnya ditulis menggunakan bahasa pemrograman PHP. Namun, berkat teknologi chromium, aplikasi dapat dijalankan di desktop dan bertingkah seperti aplikasi desktop pada umumnya
