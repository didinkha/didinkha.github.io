Project Description

Bulan Juli 2019 - saat ini saya bekerja sebagai web programmer di Yayasan Indonesia Mengajar. Saya bertanggung jawab untuk menangani website Indonesia Mengajar dan pengadaan sejumlah sistem informasi yang mendukung visi dan misi organisasi. Website ini memiliki fitur di antaranya: pengelolaan kabar berita, langganan informasi magang, email otomatis, dll. Untuk informasi lebih lanjut silahkan kunjungi tautan https://www.indonesiamengajar.org
