Project Description

Bulan Juli 2019 - saat ini saya bekerja sebagai web programmer di Yayasan Indonesia Mengajar. Saya bertanggung jawab untuk menangani website Indonesia Mengajar dan pengadaan sejumlah sistem informasi yang mendukung visi dan misi organisasi. Salah satu sistem informasi yang telah dikembangkan adalah sistem rekrutmen untuk para relawan yang hendak mendaftarkan diri sebagai relawan Pengajar Muda. Sistem ini dapat diakses pada tautan http://pm.indonesiamengajar.org
